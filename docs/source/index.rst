=========
CaliBrain
=========

CaliBrain is a scientific Python package for simulation-based uncertainty
estimation and calibration in EEG/MEG inverse source imaging.

The documentation is organized into primary sections. Detailed pages are
nested inside each section and appear in the sidebar for that section.
