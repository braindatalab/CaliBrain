Scientific References
=====================


This page collects references relevant to the methods and software ecosystem
used by CaliBrain. Add precise paper-specific references as the corresponding
methods are finalized for publication.

Related software
----------------


- MNE-Python: EEG/MEG processing, forward modeling, and source-space utilities.
- NumPy and SciPy: numerical linear algebra and scientific computing.
- scikit-learn: estimator conventions, parameter grids, cross-validation, and
  isotonic regression.
- POT: optimal transport utilities used for EMD-style metrics.

Methodological topics
---------------------


- EEG/MEG forward modeling and inverse source imaging.
- Sparse Bayesian learning and Gamma-MAP source estimation.
- Minimum-norm and Bayesian minimum-norm inverse estimation.
- eLORETA-style distributed source reconstruction.
- Isotonic regression for monotone calibration maps.
- Coverage calibration of credible intervals and ellipsoids.
