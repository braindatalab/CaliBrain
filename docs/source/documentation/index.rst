Documentation overview
======================

This section provides the main user-facing documentation for CaliBrain. It is
organized around the current workflow for uncertainty estimation and
calibration in EEG/MEG inverse source imaging.

If you have not installed CaliBrain yet, start with the pages in
`Installation <../installation/index.html>`_. Runnable entry points are
collected in the `Tutorials <../tutorials/index.html>`_ section.

The documentation is divided into a small number of main parts:

- `Conceptual Overview <conceptual_overview.html>`_ introduces the scientific problem, the package
  scope, and the main workflow stages.
- `Glossary <glossary.html>`_ provides short definitions of CaliBrain-specific
  terminology and general neuroimaging concepts used in the documentation.
- `Tutorials <../tutorials/index.html>`_ provide runnable, ordered guides for
  the core CaliBrain analysis steps.
- `Example Dataset <../datasets/index.html>`_ describes the dataset resources
  used across the documentation.
- `How to Cite CaliBrain <../cite/index.html>`_ provides citation information
  for software and scientific use.

The remaining pages in this section are listed in the sidebar and can be used
to navigate directly to a specific topic.
