Papers Citing CaliBrain
=======================

This page is intended to list publications, preprints, theses, and technical
reports that use or discuss CaliBrain.

Current status
--------------

No curated citation list is currently maintained in this documentation. Add
entries here when papers using CaliBrain become available.

Suggested entry format
----------------------

Use a consistent bibliographic format:

.. code-block:: text

   Author, A., Author, B., & Author, C. (Year). Title. Journal or venue.
   DOI or URL.

For each entry, include a short note describing how CaliBrain was used, for
example:

- uncertainty calibration benchmark;
- M/EEG inverse-solver comparison;
- simulation study;
- calibration metric or visualization method;
- workflow or software dependency.

Adding a paper
--------------

When adding a paper, verify:

- the paper actually used CaliBrain or directly discusses it;
- the citation metadata is complete;
- links use persistent identifiers when available;
- the CaliBrain version or commit hash is recorded when reported by the paper.
