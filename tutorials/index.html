Tutorials#

The tutorials page is the main runnable learning surface of the documentation. It follows a deliberate progression:

  1. scientific motivation and calibration concepts;

  2. source, leadfield, and sensor simulation;

  3. inverse source estimation;

  4. uncertainty estimation and calibration;

  5. metric evaluation, end-to-end calibration, and workflow orchestration.

The first tutorials therefore form a coherent path from simulation inputs to calibration outputs before introducing the higher-level DataGenerator workflow wrapper.
